### Bit operation
* for `&`: bitwise `and` of x and y
* for `|`: bitwise `or` of x and y 
* for `^`: bitwise `exclusive or` of x and y
* for `~`: the bits of x `inverted`
* 

```python
def time_two(n):
	return n << 1
def div_two(n):
	return n >> 1
def time_two_power(n):
	return n << m
def div_two_power(n):
	return n >> m
```

```python
def is_odd_number(n):
	return (n & 1) == 1
```
